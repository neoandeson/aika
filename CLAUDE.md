# Aika — AI Kit Awareness

## Session Resume
- At start of every session, read `.aika/context.md` for auto-tracked activity (commands, files, stats)
- Check the active plan in `docs/superpowers/plans/` — checkboxes (`- [x]` / `- [ ]`) show task progress
- `.aika/context.md` is auto-generated by hooks on session end — it shows what happened, not what to do next

## Session Save
- Before ending a session (or when user says pause/stop):
  - Update plan checkboxes in `docs/superpowers/plans/` to reflect completed steps
  - Add a "Next Up" section to `.aika/context.md` with what to work on next (this part is manual)
- The hook auto-update will preserve the activity log; manual notes go at the bottom

## How Context Tracking Works
- Hooks fire during Claude Code usage → events logged to `.aika/journal.jsonl`
- On session end (Stop/SubagentStop) → `on-stop.mjs` auto-generates `.aika/context.md` from the journal
- `.aika/context.md` is git-tracked (rest of `.aika/` is gitignored)
